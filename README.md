# DSA Algorithms Practice

This repository contains implementations of various Data Structures and Algorithms (DSA) for practice purposes. The code includes classic algorithms such as Dijkstra's, Kruskal's, Huffman Coding, Fractional Knapsack, Merge Sort, and more.

## Purpose
This repository is intended for learning and practicing fundamental DSA concepts. Each file demonstrates a specific algorithm with clear and concise code.

## Usage
Feel free to use, modify, and experiment with the code for educational purposes.
